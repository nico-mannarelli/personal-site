A comprehensive framework for developing quantum machine learning models. Features include:

- Hybrid classical-quantum neural network architectures
- Optimized parameter-shift gradients for training
- Support for various quantum feature maps
- Integration with classical ML pipelines

Built to bridge the gap between quantum computing research and practical ML applications.
